Initial Setup

Getting Started

Quick steps to get YOU music running in your server.

  1. Click "Add to Discord" and select your server
  2. Grant the required permissions
  3. Use /help to see available commands
  4. Join a voice channel and start playing music!

Required Permissions

Essential Permissions

Make sure the bot has these permissions for full functionality:

  1. View Channels & Messages
  2. Send Messages & Embeds
  3. Connect to Voice Channels
  4. Speak in Voice Channels
  5. Use Voice Activity
  6. Add Reactions
  7. Attach Files (for lyrics)

Connection Issues

Common Connection Problems

Solutions for connection-related issues:

  1. Verify bot has proper permissions
  2. Check server region compatibility
  3. Ensure stable internet connection
  4. Try reconnecting the bot
  5. Check voice channel user limit

Audio Problems

Audio Quality Issues

Improve your audio experience:

  1. Use high-quality audio sources
  2. Check server region for optimal connection
  3. Adjust volume levels appropriately
  4. Verify voice channel bitrate settings
  5. Monitor system resource usage

Platform-Specific

YouTube Issues

Common YouTube playback problems and solutions:

  1. Video unavailable in your region - Try using a different version
  2. Age-restricted content - Bot cannot play these videos
  3. Private videos - Ensure video is public or unlisted
  4. Playlist errors - Try adding songs individually
  5. Long videos - Maximum length is 3 hours

Spotify Integration

Troubleshooting Spotify playback:

  1. Ensure playlist/track is public
  2. Wait for playlist processing on first load
  3. Check if track is available on YouTube
  4. Use track title instead of Spotify link if issues persist
  5. Verify Spotify integration is enabled

Local Files

Guidelines for local file playback:

  1. Maximum file size: 25MB
  2. Supported formats: MP3, WAV, OGG
  3. Files must not be corrupted
  4. Proper metadata recommended
  5. Upload in supported channel

Command Issues

Common Command Problems

Solutions for command-related issues:

  1. Verify correct command syntax and spelling
  2. Check if bot has required permissions
  3. Ensure you're using commands in allowed channels
  4. Wait a few seconds between commands
  5. Check if bot is responding to other commands

Permission Requirements

Required permissions for commands:

Music Commands
Basic Commands Send Messages, View Channel
Voice Commands Connect, Speak, Use Voice Activity
Admin Commands Manage Channels, Manage Messages

Command Cooldowns

Understanding command rate limits:

  1. Basic commands: 3 second cooldown
  2. Queue management: 5 second cooldown
  3. Effect commands: 10 second cooldown
  4. Admin commands: 30 second cooldown
  5. Playlist loading: 60 second cooldown
