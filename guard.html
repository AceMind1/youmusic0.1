YOU guard - Protection

Server protection against harassment and inappropriate content

v0.1.0 February 6, 2025
Release

  • Anti-Harassment System

    Core Features:

    • Content filtering
    • Automatic message moderation
    • Warning system with configurable thresholds
    • Timeout enforcement
    • Detailed logging of incidents

  • Warning System

    Warning Commands
    /checkwarnings View warnings for a user
    /clearwarnings Clear user warnings (Admin)

    Warning Features:

    • Progressive warning system
    • 30-day warning expiry
    • Automatic timeout after max warnings
    • Warning history tracking

  • Content Filtering

    Protected Content Types:
    • Inappropriate language
    • Hate speech
    • Discriminatory content
    • NSFW content
    • Spam detection

    Filter Features:
    • Real-time message scanning
    • Pattern recognition
    • Unicode character analysis
    • Context-aware filtering
    • Bypass attempt detection

  • Moderation Tools

    Administrative Features:
    • Detailed incident logging
    • Customizable filter settings
    • Moderation action tracking
    • User history monitoring
    • Auto-moderation rules
